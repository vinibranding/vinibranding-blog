# 마크다운 기반 블로그 시스템 도입

안녕하세요! Vini's Branding Lab입니다.

이제 **마크다운(Markdown)** 기반의 시스템이 도입되어 훨씬 간편하게 글을 올릴 수 있게 되었습니다. 

## 1. 어떻게 작성하나요?
복잡한 관리자 페이지에 접속할 필요 없이, 인공지능 어시스턴트에게 **"이런 제목과 내용으로 글을 올려줘"**라고 요청하기만 하면 됩니다.
AI가 자동으로 마크다운 형식으로 변환하여 블로그에 포스팅을 반영합니다.

## 2. 이미지도 넣을 수 있나요?
네, 이미지 URL을 주시거나, AI에게 관련 이미지를 생성/검색해 달라고 요청할 수 있습니다!

## 3. 포맷팅
*글씨 기울임*, **굵게**, 그리고 이렇게 목록도 작성할 수 있습니다.

- 직무 역량 중심의 자소서
- 퍼스널 컬러와 비즈니스 캐주얼
- 대기업 공채 트렌드

앞으로 다양한 콘텐츠를 손쉽게 추가해 보세요!
